🔢 Tribonacci Series Challenge

A JavaScript-based solution to generate the Tribonacci sequence—a variation of the Fibonacci series where each term is the sum of the preceding three numbers. This challenge showcases iterative logic, array manipulation, and clean modular code.

✨ Features
📈 Generates Tribonacci sequences of any length

🔁 Uses iterative logic for performance and clarity

🧩 Modular structure for easy reuse and extension
